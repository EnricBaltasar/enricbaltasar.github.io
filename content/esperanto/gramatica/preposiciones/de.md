<!--more-->La preposición <strong>DE</strong> en esperanto es la traducción más frecuente de <span style="text-decoration: underline;"><em>de</em></span> en castellano. Por ello, cuando dudes, utiliza esta.

 
<h2 id="pertenencia-de"><span style="color: #ff6600;"><strong>PERTENENCIA (de)</strong>
</span></h2>
<pre>» <strong>La libro <span style="text-decoration: underline;">de</span> Juán</strong> <span style="color: #808080;">(el libro <span style="text-decoration: underline;">de</span> Juán)</span>
» <strong>Verkoj <span style="text-decoration: underline;">de</span> la aŭtoro</strong> <span style="color: #808080;">(obras <span style="text-decoration: underline;">de</span>l autor)</span>
» <strong>Paĝoj <span style="text-decoration: underline;">de</span> la libro</strong> <span style="color: #808080;">(páginas <span style="text-decoration: underline;">de</span>l libro)</span>
» <strong>La kanto <span style="text-decoration: underline;">de</span> la birdoj</strong> <span style="color: #808080;">(el canto <span style="text-decoration: underline;">de</span> los pájaros)</span>
» <strong>Ricevi helpon <span style="text-decoration: underline;">de</span> la instruisto</strong> <span style="color: #808080;">(recibir ayuda <span style="text-decoration: underline;">de</span>l profesor)</span>
» <strong>Instruisto <span style="text-decoration: underline;">de</span> matematiko</strong> <span style="color: #808080;">(profesor <span style="text-decoration: underline;">de</span> matemáticas)</span>
» <strong>Membro <span style="text-decoration: underline;">de</span> la asocio</strong> <span style="color: #808080;">(miembro <span style="text-decoration: underline;">de</span> la asociación)</span>
» <strong>Ili estas bestoj <span style="text-decoration: underline;">de</span> malsamaj specoj</strong> <span style="color: #808080;">(son animales <span style="text-decoration: underline;">de</span> diferentes especiales)</span>
» <strong>Mi partoprenas kurson <span style="text-decoration: underline;">de</span> Esperanto</strong> <span style="color: #808080;">(hago un curso <span style="text-decoration: underline;">de</span> esperanto)</span></pre>
 
<h2 id="comienzo-de-desde-a-partir-de"><span style="color: #ff6600;"><strong>COMIENZO (de, desde, a partir de)</strong>
</span></h2>
<pre>» <strong>Iri <span style="text-decoration: underline;">de</span> Parizo</strong> <span style="color: #808080;">(ir <span style="text-decoration: underline;">de</span> París / ir <span style="text-decoration: underline;">desde</span> París)</span>
» <strong>Forkuri <span style="text-decoration: underline;">de</span> mi</strong> <span style="color: #808080;">(huir <span style="text-decoration: underline;">de</span> mí)</span>
» <strong>De hodiaŭ</strong> <span style="color: #808080;">(<span style="text-decoration: underline;">desde</span> hoy)</span>
» <strong>De morgaŭ mi ekmanĝos malpli</strong> <span style="color: #808080;">(<span style="text-decoration: underline;">a partir de</span> mañana empezaré a comer menos)</span>
» <strong>De la sesa (horo)</strong> <span style="color: #808080;">(<span style="text-decoration: underline;">desde</span> las siete)</span>
</pre>
 
<h2 id="separacion-de"><span style="color: #ff6600;"><strong>SEPARACIÓN (de)</strong>
</span></h2>
