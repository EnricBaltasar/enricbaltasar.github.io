¡Tendrás a tu suegra hasta en la sopa con lo fácil que será mencionarla! El prefijo <span style="color: #ff6600;"><strong>bo-</strong></span> sirve para indicar parentesco familiar:

<pre><strong>   AVOJ</strong>                     <strong>BOAVOJ</strong>
 <span style="color: #808080;">(abuelos)</span>            <span style="color: #808080;">(abuelos políticos)</span>
    |                         |
  <strong>PATROJ</strong>                   <strong>BOPATROJ</strong>
 <span style="color: #808080;">(padres) </span>                <span style="color: #808080;">(suegros)</span>
    |                         |
   / \                       / \
 <strong><span style="color: #ff6600;">MI</span></strong>  <strong>FRATOJ</strong>              <strong><span style="color: #ff6600;">EDZO</span></strong>   <strong>BOFRATOJ</strong>
<span style="color: #808080;">(yo)</span> <span style="color: #808080;">(hermanos)</span>      <span style="color: #808080;">(cónyuge)</span>  <span style="color: #808080;">(cuñados)</span></pre>

Este prefijo se puede utilizar con más palabras, pero prácticamente solo se utiliza con las de la tabla anterior.

<strong>Nota:</strong> es importante aclarar que en un principio se utilizaba solo para parejas casadas, pero actualmente se usa también un poco para novios (y no es el único idioma en el que pasa esto, pues los tiempos cambian).
