La preposición <strong>EL</strong> en esperanto.

 
<h2 id="origen-de"><span style="color: #ff6600;"><strong>ORIGEN (de)</strong>
</span></h2>
<pre>» <strong><span style="text-decoration: underline;">El</span> kie vi estas/venas?</strong> <span style="color: #808080;">(¿<span style="text-decoration: underline;">De</span> dónde eres/vienes?)</span>
» <strong>Trinki <span style="text-decoration: underline;">el</span> la glaso</strong> <span style="color: #808080;">(beber <span style="text-decoration: underline;">de</span>l vaso)</span>
» <strong>Fari <span style="text-decoration: underline;">el</span> vi kaĉon</strong> <span style="color: #808080;">(hacer <span style="text-decoration: underline;">de</span> ti papilla)</span>
» <strong><span style="text-decoration: underline;">El</span> la infano iĝis ĉampiono</strong> <span style="color: #808080;">(<span style="text-decoration: underline;">de</span>l niño se hizo un campeón)</span>
» <strong>Li vekiĝis <span style="text-decoration: underline;">el</span> longa dormo</strong> <span style="color: #808080;">(se despertó <span style="text-decoration: underline;">de</span> un largo sueño)</span>
» <strong>Vi faris <span style="text-decoration: underline;">el</span> mi veran atleton</strong> <span style="color: #808080;">(has hecho <span style="text-decoration: underline;">de</span> mí todo un atleta)</span>
» <strong>Mi venas <span style="text-decoration: underline;">el</span> Italio</strong> <span style="color: #808080;">(vengo <span style="text-decoration: underline;">de</span> Italia)</span>
» <strong>Ni kuiris omleton <span style="text-decoration: underline;">el</span> spinacoj</strong> <span style="color: #808080;">(hemos hecho una tortilla <span style="text-decoration: underline;">de</span> espinacas)</span>
» <strong>La kanto estas <span style="text-decoration: underline;">el</span> la 70-aj jaroj</strong> <span style="color: #808080;">(la canción es <span style="text-decoration: underline;">de</span> los años 70)</span>
» <strong>La kukon mi faris <span style="text-decoration: underline;">el</span> fromaĝo</strong> <span style="color: #808080;">(el pastel lo he hecho <span style="text-decoration: underline;">de</span> queso)</span></pre>
 
<h2 id="movimiento-afuera-de"><span style="color: #ff6600;"><strong>MOVIMIENTO AFUERA (de)</strong>
</span></h2>
<pre>» <strong>Mi eliris <span style="text-decoration: underline;">el</span> la domo</strong> <span style="color: #808080;">(he salido <span style="text-decoration: underline;">de</span> casa)</span></pre>
 
<h2 id="elemento-de-grupo-de"><span style="color: #ff6600;"><strong>ELEMENTO DE GRUPO (de)</strong>
</span></h2>
Algunos de estos casos coinciden con el <a href="/superlativo-esperanto/">superlativo</a> (el más alto <span style="text-decoration: underline;">de</span> todos; el peor <span style="text-decoration: underline;">de</span> todos).
<pre>» <strong>Vi estas unu <span style="text-decoration: underline;">el</span> ili</strong>
<span style="color: #808080;">(eres uno <span style="text-decoration: underline;">de</span> ellos)</span>
» <strong><span style="text-decoration: underline;">El</span> miaj amikoj vi estas la plej riĉa</strong>
<span style="color: #808080;">(<span style="text-decoration: underline;">de</span> mis amigos, eres el más rico)</span>
» <strong><span style="text-decoration: underline;">El</span> la tri leteroj unu estis adresita al vi</strong>
<span style="color: #808080;">(<span style="text-decoration: underline;">de</span> las tres cartas, una iba dirigida a ti)</span>
» <strong>Minuto konsistas <span style="text-decoration: underline;">el</span> sesdek sekundoj</strong>
<span style="color: #808080;">(un minuto consiste en sesenta segundos)</span>
» <strong>Tie staras la plej bela katedralo <span style="text-decoration: underline;">el</span> la mondo</strong>
<span style="color: #808080;">(allí está la catedral más bella <span style="text-decoration: underline;">de</span>l mundo)</span></pre>
