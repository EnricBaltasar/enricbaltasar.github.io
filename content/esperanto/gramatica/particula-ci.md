<!--more-->La partícula <span style="color: #ff6600;"><strong>ĉi</strong></span> indica proximidad:

<pre><strong>Tie</strong> <span style="color: #808080;">(ahí, allí, allá)</span> → <strong><span style="text-decoration: underline;">Ĉi</span> tie</strong> <span style="color: #808080;">(aquí)</span>
<strong>Tiu</strong> <span style="color: #808080;">(ese, aquel)</span> → <strong><span style="text-decoration: underline;">Ĉi</span> tiu</strong> <span style="color: #808080;">(este)</span>
<strong>Tio</strong> <span style="color: #808080;">(eso)</span> → <strong><span style="text-decoration: underline;">Ĉi</span> tio</strong> <span style="color: #808080;">(esto)</span></pre>

Como ves, se pone delante de un correlativo (ĉi + tie, tiu...). Pero también se puede poner detrás:

<pre><strong>Tie</strong> <span style="color: #808080;">(ahí, allí, allá)</span> → <strong>Tie <span style="text-decoration: underline;">ĉi</span></strong> <span style="color: #808080;">(aquí)</span>
<strong>Tiu</strong> <span style="color: #808080;">(ese, aquel)</span> → <strong>Tiu <span style="text-decoration: underline;">ĉi</span></strong> <span style="color: #808080;">(este)</span>
<strong>Tio</strong> <span style="color: #808080;">(eso)</span> → <strong>Tio <span style="text-decoration: underline;">ĉi</span></strong> <span style="color: #808080;">(esto)</span></pre>

Hasta aquí deberías leer si no habías aprendido nunca <em>ĉi</em>. Pero, si ya habías tenido el gusto de conocerlo, te invito a seguir leyendo:

 

<h2 id="ci-correlativo"><span style="color: #ff6600;"># ĈI + CORRELATIVO</span></h2>

<em>Ĉi</em> se puede usar también junto a los otros correlativos de <em>ti-</em>. Veamos algunos ejemplos:

<pre><strong>Tiel</strong> <span style="color: #808080;">(así)</span> → <strong>Ĉi tiel</strong> <span style="color: #808080;">(de esta manera, de este modo)</span>
<strong>Tio</strong> <span style="color: #808080;">(eso)</span> → <strong>Ĉi tio</strong> <span style="color: #808080;">(todo esto, todo eso)</span>
<strong>Tiam</strong> <span style="color: #808080;">(entonces)</span> → <strong>Ĉi tiam</strong> <span style="color: #808080;">(en este momento, en ese entonces)
</span><strong>Ĉio</strong><span style="color: #808080;"> (todo)</span> → <strong>Ĉio ĉi</strong><span style="color: #808080;"> (todo esto, todo eso)
</span><strong>Ĉiu</strong><span style="color: #808080;"> (cada uno, todo/a)</span> → <strong>Ĉiun ĉi tempon</strong><span style="color: #808080;"> (todo este tiempo, todo ese tiempo)
</span></pre>

 

<h2 id="ci-tiu-palabra"><span style="color: #ff6600;"># ĈI TIU + PALABRA
</span></h2>

Cuando se den casos como este, podemos abreviar suprimiendo el correlativo:

<pre><strong>Ĉi <span style="text-decoration: underline;">tiu</span> libro</strong> <span style="color: #808080;">(este libro)</span> → <strong>Ĉi libro</strong> <span style="color: #808080;">(este libro)</span>
<strong>Ĉi <span style="text-decoration: underline;">tiuj</span> libroj</strong> <span style="color: #808080;">(estos libros)</span> → <strong>Ĉi libroj</strong> <span style="color: #808080;">(estos libros)</span>

<strong>Ĉi <span style="text-decoration: underline;">tiu</span> bela libro</strong> <span style="color: #808080;">(este libro bonito)</span> → <strong>Ĉi libro</strong> <span style="color: #808080;">(este libro)</span>
<strong>Ĉi <span style="text-decoration: underline;">tiuj</span> belaj libroj</strong> <span style="color: #808080;">(estos libros bonitos)</span> → <strong>Ĉi belaj libroj</strong> <span style="color: #808080;"><strong>(estos libros bonitos)</strong>

</span><strong>Ĉu vi vidis ĉi <span style="text-decoration: underline;">tiujn</span> aliajn?</strong><span style="color: #808080;"> (¿Viste estos otros?)</span> → <strong>Ĉu vi vidis ĉi aliajn?</strong><span style="color: #808080;"> (¿Viste estos otros?)
</span></pre>

 

<h2 id="ci-pegado"><span style="color: #ff6600;"># ĈI PEGADO
</span></h2>

También podemos pegar <em>ĉi</em> a un adjetivo o adverbio con un guion, dando ello un sentido distinto:

<pre><strong>Sube estas manĝaĵo</strong> <span style="color: #808080;">(debajo hay comida)</span>
→ <strong>Ĉi-sube estas manĝaĵo</strong> <span style="color: #808080;">(aquí debajo hay comida)</span>

<strong>Rivera akvo</strong> <span style="color: #808080;">(agua de río)</span>
→ <strong>Ĉi-rivera akvo</strong> <span style="color: #808080;">(agua de este río)</span></pre>

 

<h2 id="faltas"><span style="color: #ff6600;"># FALTAS
</span></h2>

Un error que cometen algunos es escribir:

<pre><strong><span style="color: #ff0000;">Ĉi-tiu</span></strong> /<strong> <span style="color: #ff0000;">Tiu-ĉi</span></strong></pre>

En vez de:

<pre><span style="color: #008000;"><strong>Ĉi tiu</strong></span> / <span style="color: #008000;"><strong>Tiu ĉi</strong></span></pre>
