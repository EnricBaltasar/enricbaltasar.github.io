La instalación de Python trae consigo un intérprete de comandos, normalmente llamado <em>terminal</em> o <em>consola de Pytho</em>n. Y puede que no se necesite más, pero hay uno mucho mejor: <strong>bpython</strong>.<!--more-->

Las ventajas de bpython son, más que nada:
<ul>
	<li><strong>Marcado de sintaxis</strong> con colores.</li>
	<li><strong>Autocompletado</strong>.</li>
	<li><strong>Soporte para Python 2 y Python 3</strong> al mismo tiempo, lo cual es genial para los que empiezan a aprender y no saben discriminar las versiones, o para probar rápidamente fragmentos de código que se encuentran.</li>
</ul>
Por supuesto, es <strong>multiplataforma</strong>: GNU/Linux, Windows, OS X y BSD.

En cuanto a la licencia, es MIT.

<hr />

 
<h2 id="instalacion">INSTALACIÓN</h2>
Ejecutamos esta línea de comandos en la consola:

<code>sudo apt-get install bpython</code>

 

Luego, para iniciar el programa, escribimos:

<code>bpython</code>

 

Si tienes algún problema con la instalación, en la página <a href="http://www.bpython-interpreter.org/downloads.html" target="_blank" rel="noopener noreferrer">Bpython » Downloads</a> encontrarás la solución.

 

<hr />

 
<h2 id="uso">USO</h2>
Algunas cosas importantes:
<ul>
	<li>La función de autocompletar se utiliza con las teclas TAB y ENTER.</li>
	<li>Para limpiar la consola, usamos CTRL + L.</li>
	<li>Para salir a la consola del sistema, escribimos <em>quit()</em>. Para cerrarlo todo empleamos <em>exit</em>.</li>
	<li>En la página <a href="http://docs.bpython-interpreter.org/configuration.html" target="_blank" rel="noopener noreferrer">Configuración</a> encontrarás más recursos, por ejemplo para cambiar el color de fondo del entorno.</li>
	<li>Por la compatibilidad de las versiones 2 y 3 de Python, en algún caso contado obtendrás un error.</li>
</ul>
